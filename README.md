# TheaterManagement
My Final Project design for CSc11300. A Python based backend that handles typical functionality for a theater such as listing new titles, setting movie ratings, available seat counts, selling tickets and snacks, etc.

Originally in Replit, this project was a collaboration between myself and JPlatt, my teammate, for our Programming Language Final Project/Assignment.
Terminal based inputs from the user allows for the creation of new movies to be added at varying times with their required data and the purchase of tickets and snacks for users.
